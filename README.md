## 注意
js的数字范围是`Number.MIN_SAFE_INTEGER`(-9007199254740991)     
 ~ `Number.MAX_SAFE_INTEGER`(9007199254740991)           
验证的时候超过这个范围就最好转成字符串在进行校验
